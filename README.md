# Oreshkes

Oreshkes - top-down roguelike на Unity про короткие напряжённые забеги, процедурные комнаты, оружие, модификаторы и бой с выразительной аркадной подачей.

Проект собирается вокруг ощущения быстрого прохода по опасному техногенному пространству: игрок входит в комнату, читает угрозы, собирает ресурсы, усиливается через предметы и модификаторы, а затем продвигается глубже к более плотным столкновениям и боссу.

## About

Oreshkes - это небольшой, но плотный roguelike-проект с фокусом на темп, читаемость боя и наращивание силы во время забега. В основе - комнаты с врагами и наградами, несколько типов оружия, pickups, магазин, система модификаторов и отдельная boss-сцена с более постановочным столкновением.

Проект сейчас находится в активной разработке: механики, визуальный стиль, UI и структура уровней продолжают шлифоваться.

## Gameplay Focus

- Динамичный top-down бой с ближним и дальним оружием.
- Процедурная структура комнат и переходов.
- Модификаторы игрока и оружия, которые меняют ощущение забега.
- Pickups, ресурсы и внутриигровой магазин.
- HUD, меню, настройки и визуальные эффекты, собранные под цельный игровой поток.
- Босс-экскаватор как отдельная кульминация текущего прототипа.

## Release

Актуальные игровые билды лучше выкладывать через [GitHub Releases](https://github.com/wssdozh/oreshkes/releases).

Хороший релиз для проекта выглядит так:

- Tag: `v0.1.0`, `v0.2.0`, `v0.3.0`.
- Title: `Oreshkes Prototype v0.1.0`.
- Asset: архив билда, например `Oreshkes_Windows_v0.1.0.zip`.
- Notes: 3-5 коротких строк о главном в версии, без технической простыни.

Пример структуры release notes:

```md
## Oreshkes Prototype v0.1.0

Первый публичный билд top-down roguelike-прототипа.

### In This Build
- Процедурные комнаты и боевые столкновения.
- Несколько типов оружия и модификаторов.
- Pickups, магазин, HUD и boss encounter.

### Download
- Windows: `Oreshkes_Windows_v0.1.0.zip`
```
