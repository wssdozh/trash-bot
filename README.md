# Oreshkes

Oreshkes - top-down roguelike на Unity про короткие забеги по опасным комнатам, оружие, модификаторы и драки, где быстро становится тесно.

Игрок заходит в комнату, разбирается с тем, что там происходит, собирает ресурсы, находит усиления и идёт дальше. Забег постепенно набирает вес: врагов больше, ошибок прощают меньше, билд начинает ощущаться по-другому.

## Что это

Это прототип roguelike с акцентом на темп и понятный бой. Внутри уже есть процедурные комнаты, несколько типов оружия, pickups, магазин, модификаторы, HUD, меню и отдельная встреча с боссом-экскаватором.

Проект ещё в разработке. Сейчас главное - довести ощущение забега: чтобы комнаты читались быстро, оружие отличалось не только цифрами, а усиления реально меняли то, как играешь.

## Что уже есть

- Top-down бой с ближним и дальним оружием.
- Комнаты, переходы и структура забега.
- Модификаторы игрока и оружия.
- Pickups, ресурсы и магазин.
- HUD, меню, настройки и визуальные эффекты.
- Босс-экскаватор как большая проверка текущего билда.

## Технологии

- Unity `6000.2.8f1`
- Universal Render Pipeline
- Unity Input System
- AI Navigation
- TextMeshPro / Unity UI
- Git LFS для крупных ассетов
