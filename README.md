# Oreshkes

Oreshkes - top-down action на Unity про короткие напряжённые забеги, процедурные комнаты, оружие, модификаторы и бой с выразительной аркадной подачей.

Проект собирается вокруг ощущения быстрого прохода по опасному техногенному пространству: игрок входит в комнату, читает угрозы, собирает ресурсы, усиливается через предметы и модификаторы, а затем продвигается глубже к более плотным столкновениям и боссу.

## About

Oreshkes - это небольшой, но плотный action-проект с фокусом на темп, читаемость боя и наращивание силы во время забега. В основе - комнаты с врагами и наградами, несколько типов оружия, pickups, магазин, система модификаторов и отдельная boss-сцена с более постановочным столкновением.

Проект сейчас находится в активной разработке: механики, визуальный стиль, UI и структура уровней продолжают шлифоваться.

## Gameplay Focus

- Динамичный top-down бой с ближним и дальним оружием.
- Процедурная структура комнат и переходов.
- Модификаторы игрока и оружия, которые меняют ощущение забега.
- Pickups, ресурсы и внутриигровой магазин.
- HUD, меню, настройки и визуальные эффекты, собранные под цельный игровой поток.
- Босс-экскаватор как отдельная кульминация текущего прототипа.

## Tech

- Unity `6000.2.8f1`
- Universal Render Pipeline
- Unity Input System
- AI Navigation
- TextMeshPro / Unity UI
- Git LFS для крупных ассетов

## Run Locally

1. Установить Unity `6000.2.8f1`.
2. Клонировать репозиторий и подтянуть LFS-файлы:

   ```powershell
   git lfs install
   git clone https://github.com/wssdozh/oreshkes.git
   cd oreshkes
   git lfs pull
   ```

3. Открыть проект через Unity Hub.
4. Запустить сцену `Assets/Scenes/MainMenuScene.unity`.
