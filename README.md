# Oreshkes

Oreshkes - top-down roguelike на Unity про короткие забеги по опасным комнатам. Игрок подбирает оружие, собирает ресурсы, усиливается через модификаторы и постепенно пробивается к более плотным боям.

Изначально это был дипломный проект. Сейчас он оформлен как игровой прототип и портфолио-репозиторий: главная цель - показать быстрый, читаемый забег, где оружие ощущается по-разному, а найденные усиления реально меняют стиль игры.

## Платформа

- PC / Windows
- Unity build

## Что реализовано

- Процедурная структура комнат и переходов.
- Top-down бой с ближним и дальним оружием.
- Несколько типов оружия и модификаторы для них.
- Pickups, ресурсы и внутриигровой магазин.
- HUD, меню, настройки и визуальная обратная связь.
- Boss encounter с экскаватором.

## Моя роль

Я делал Oreshkes как дипломный проект и за 6 месяцев довёл его до состояния игрового прототипа: с рабочим забегом, боем, прогрессией, UI и boss encounter.
